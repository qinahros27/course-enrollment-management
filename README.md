# Course Enrollment Management App
An app that can add , edit , delete the course , and connecting with MongoDB. 

## Technologies 
- Javascript
- Nodejs
- Pug template

## Project structure
```
.
└───CourseEnrollmentManagement
    └───.vscode
    └───node_modules
    └───views
        └───addcourse.pug
        └───courses.pug
        └───deletecourse.pug
        └───detailcourse.pug
        └───editcourse.pug
    └───course.js
    └───index.js
    └───package-lock.json
    └───routes.js
    └───README.md
```
## Getting started
Clone the respository from github: ```git clone```
